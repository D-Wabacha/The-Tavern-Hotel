# Royal Tavern Hotel

Production-ready landing page for Royal Tavern Hotel — Othaya, Nyeri County, Kenya.

## Run locally

```bash
npm install
npm run dev
```

Open http://localhost:5173

## Build

```bash
npm run build
npm run preview
```

## Deploy

Any static host works (Vercel, Netlify, Cloudflare Pages, GitHub Pages).
The `dist/` folder produced by `npm run build` is what you publish.

## Replace images

Edit the `images` object at the top of `src/App.tsx`.
